# Steam Workshop listing copy (per-language descriptions)

The uploader sets `language` (default = english) + `localizations`
(`{language, title, description}` per Steam language code; Korean = "koreana").
So English-language users see the English block, Korean-language users see the
Korean block. Two standalone blocks below — first = English (default), second =
Korean. Steam BBCode: [h1]/[h2]/[h3] headings (NOT [size]), [list], [olist].

## English (default)

```bbcode
[h2]Every map gets a theme[/h2]
When a new act map appears, this mod gives it a [b]theme[/b] — and actually builds the map to match. A [b]Merchant's Run[/b] is full of shops. An [b]Elite Gauntlet[/b] is packed with elites. A [b]Sanctuary[/b] is full of campfires. A [b]Labyrinth[/b] is a maze of branching paths. You can see the whole map, so you know the theme up front and plan your route around it.

[h2]24 themes to run into[/h2]
Peaceful ones, deadly ones, rich ones, maze-like ones, and more. You never pick — each map surprises you with a theme. Easier themes tend to show up early; the nastier ones show up deeper in the run.

[h2]The events match the theme too[/h2]
The mystery [?] rooms lean toward the mood as well: a scary map gives you more dangerous events, a market map more shop events, a restful map more healing ones. Every event can still appear — the odds just tilt toward the theme.

[h2]Good to know[/h2]
Every map also has a difficulty rank — [b]I[/b], [b]II[/b], or [b]III[/b] — shown right next to its name. The higher the rank, the harder the theme hits.

[h2]Want to try a specific theme?[/h2]
Turn on the dev console and type [b]conceptmap list[/b] to see them all, or [b]conceptmap <name>[/b] to force one. Otherwise you just get a fresh surprise theme on every map.

[h2]Compatibility[/h2]
[list]
[*][b]Co-op:[/b] everyone gets the same map — just run the same version.
[*][b]Don't run it alongside the Random Map mod[/b] — they both change map nodes.
[/list]

[h2]Source[/h2]
[url=https://github.com/ing-gom/sts2-concept-map]github.com/ing-gom/sts2-concept-map[/url] (MIT) · Sister mod: [url=https://github.com/ing-gom/sts2-random-map]Random Map[/url] · Built on Slay the Spire 2 by MegaCrit.
```

## Korean (koreana)

```bbcode
[h2]맵마다 테마가 생깁니다[/h2]
새 막의 지도가 나오면 이 모드가 맵에 [b]테마[/b]를 하나 정하고, 맵을 실제로 그 테마대로 만듭니다. [b]상인의 길[/b]은 상점 천지, [b]정예 시련장[/b]은 정예로 가득, [b]안식처[/b]는 모닥불이 잔뜩, [b]미궁[/b]은 갈림길투성이 미로. 맵이 전부 보이니 테마를 미리 확인하고 경로를 짤 수 있습니다.

[h2]마주치는 테마 24종[/h2]
편안한 곳, 위험한 곳, 부유한 곳, 미로 같은 곳 등. 직접 고르는 게 아니라 맵마다 테마가 랜덤으로 정해집니다. 쉬운 테마는 주로 앞막에, 고약한 테마는 뒤로 갈수록 나옵니다.

[h2]이벤트도 테마에 맞춰집니다[/h2]
물음표 [?] 방도 분위기 쪽으로 기웁니다. 무서운 맵엔 위험한 이벤트가, 시장 맵엔 상점 이벤트가, 편안한 맵엔 치유 이벤트가 더 자주 나옵니다. 모든 이벤트는 여전히 나올 수 있고, 확률만 테마 쪽으로 기울 뿐입니다.

[h2]알아두면 좋은 점[/h2]
맵마다 난이도 등급([b]Ⅰ[/b] · [b]Ⅱ[/b] · [b]Ⅲ[/b])이 이름 옆에 표시됩니다. 등급이 높을수록 테마가 더 강하게 적용됩니다.

[h2]특정 테마를 써보고 싶다면[/h2]
개발자 콘솔을 켜고 [b]conceptmap list[/b]로 전체 목록을 보거나 [b]conceptmap <이름>[/b]으로 원하는 테마를 강제할 수 있습니다. 평소엔 맵마다 랜덤 테마가 나옵니다.

[h2]호환성[/h2]
[list]
[*][b]협동:[/b] 전원 같은 맵을 받습니다 — 같은 버전만 쓰면 됩니다.
[*][b]Random Map 모드와 함께 쓰지 마세요[/b] — 둘 다 맵 노드를 바꿉니다.
[/list]

[h2]소스[/h2]

```
